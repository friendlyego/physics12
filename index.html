Average Acceleration
$$\vec{a}_{av} = \frac{\Delta \vec{v}}{\Delta t} = \frac{\vec{v}_f - \vec{v}_i}{t}$$
Instantaneous Acceleration

First Equation

Relative Motion

Kinematics
Relative Velocity
$$\vec{v}_{AB} = \vec{v}_A - \vec{v}_B$$
Velocity of A with respect to B
Relative Acceleration
$$\vec{a}_{AB} = \vec{a}_A - \vec{a}_B$$
Relative Displacement
$$x_{BA} = x_B - x_A$$
River Crossing
Shortest Time
$$t_{min} = \frac{d}{v_{mR}}, \quad v_x = v_R, \quad v_m = \sqrt{v_{mR}^2 + v_R^2}$$
Boat pointed perpendicular to river. Drift $= v_R \cdot t$

$$\sin\theta = \frac{v_R}{v_{mR}}, \quad v_y = \sqrt{v_{mR}^2 - v_R^2}$$

$$\vec{v}_{Rm} = \vec{v}_R - \vec{v}_m, \quad |\vec{v}_{Rm}| = \sqrt{v_R^2 + v_m^2}$$

Newton's Laws of Motion

Dynamics
Newton's 2nd Law
$$\vec{F} = \frac{d\vec{p}}{dt} = m\vec{a}$$
$F_x = ma_x$, $F_y = ma_y$, $F_z = ma_z$
Newton's 3rd Law
$$\vec{F}_{AB} = -\vec{F}_{BA}$$
Forces are equal in magnitude, opposite in direction, act on different bodies
Spring Force
$$F = -kx$$
$x$ = deformation from natural length; $k$ = spring constant
Spring Property
$$k \times \ell = \text{constant}$$
$\ell$ = natural length of spring
Spring Cut in Ratio m : n
$$k_1 = \frac{(m+n)}{m}\,k, \quad k_2 = \frac{(m+n)}{n}\,k$$
$k_1\ell_1 = k_2\ell_2 = k\ell$
Series Spring Combination
$$\frac{1}{k_{eq}} = \frac{1}{k_1} + \frac{1}{k_2} + \cdots$$
Parallel Spring Combination
$$k_{eq} = k_1 + k_2 + k_3 + \cdots$$
Atwood Machine
$$a = \frac{(m_1-m_2)g}{m_1+m_2}, \quad T = \frac{2m_1 m_2\, g}{m_1+m_2}$$
Pulley Constraint (Movable Pulley)
$$V_P = \frac{V_1+V_2}{2}, \quad a_P = \frac{a_1+a_2}{2}$$
Newton's Law for System
$$\vec{F}_{ext} = m_1\vec{a}_1 + m_2\vec{a}_2 + \cdots$$
Non-Inertial Frame (Pseudo Force)
$$\vec{F}_{pseudo} = -m\vec{a}_{frame}$$
Net force in non-inertial frame = Real forces + Pseudo force

$\theta$ = angle of inclined surface with horizontal

Work Done (Variable Force)
$$W = \int \vec{F} \cdot d\vec{s}$$

Momentum–KE Relation
$$p = \sqrt{2mK}$$

Modified Work-Energy Theorem

Circular Motion

Dynamics
Angular Velocity
$$\omega = \frac{d\theta}{dt}, \quad \bar{\omega} = \frac{\Delta\theta}{\Delta t}$$
Angular Acceleration
$$\alpha = \frac{d\omega}{dt} = \omega\frac{d\omega}{d\theta}$$
Speed–Angular Velocity
$$v = r\omega, \quad \vec{v} = \vec{\omega}\times\vec{r}$$
Centripetal Acceleration
$$a_r = \frac{v^2}{r} = \omega^2 r$$
Tangential Acceleration
$$a_t = \frac{dv}{dt} = r\alpha$$
Total Acceleration
$$a = \sqrt{a_t^2 + a_r^2}$$
Radius of Curvature
$$R = \frac{v^2}{a_\perp} = \frac{mv^2}{F_\perp} = \frac{\left[1+\left(\frac{dy}{dx}\right)^2\right]^{3/2}}{\left|\frac{d^2y}{dx^2}\right|}$$
Concave Bridge (Normal Force)
$$N = mg\cos\theta + \frac{mv^2}{r}$$
Convex Bridge (Normal Force)
$$N = mg\cos\theta - \frac{mv^2}{r}$$
Safe Speed on Level Road
$$v_{safe} \leq \sqrt{\mu g r}$$
Max Angular Speed on Rotating Platform
$$\omega_{max} = \sqrt{\frac{\mu g}{r}}$$
Bending of Cyclist / Banking (Frictionless)
$$\tan\theta = \frac{v^2}{rg}$$
Banking with Friction
$$\tan\theta = \frac{v^2 - \mu rg}{rg + \mu v^2}$$
Max Speed on Banked Frictional Road
$$v_{max} = \sqrt{\frac{rg(\tan\theta + \mu)}{1 - \mu\tan\theta}}$$
Min Speed on Banked Frictional Road
$$v_{min} = \sqrt{\frac{rg(\tan\theta - \mu)}{1 + \mu\tan\theta}}$$
Centrifugal Force (Pseudo Force)
$$f = m\omega^2 r \quad \text{(outward)}$$
Effect of Earth's Rotation on g
$$N = mg - m\omega_E^2 R\cos^2\lambda$$
$\lambda$ = latitude; $N$ = apparent weight
Conical Pendulum
$$T = 2\pi\sqrt{\frac{L\cos\theta}{g}}, \quad \omega = \sqrt{\frac{g}{L\cos\theta}}$$
$T\cos\theta = mg$; $T\sin\theta = m\omega^2 r$
Min Speed for Vertical Loop (Bottom)
$$v_{min} = \sqrt{5gL}$$
For completing the circle; at top: $v_{top,min} = \sqrt{gL}$
Angular Equations of Motion ($\alpha$ = const)
$$\omega = \omega_0 + \alpha t, \quad \theta = \omega_0 t + \tfrac{1}{2}\alpha t^2, \quad \omega^2 = \omega_0^2 + 2\alpha\theta$$

Coefficient of Restitution

e=1: Elastic | e=0: Perfectly Inelastic | 0<e<1: Inelastic

Simple Harmonic Motion

Oscillations
SHM Condition
$$F = -kx \implies a = -\omega^2 x$$
General Solution
$$x = A\sin(\omega t + \phi)$$
$A$ = amplitude; $\phi$ = initial phase; $\omega$ = angular frequency
Angular Frequency & Period
$$\omega = \sqrt{\frac{k}{m}} = \frac{2\pi}{T} = 2\pi f, \quad T = 2\pi\sqrt{\frac{m}{k}}$$
Velocity
$$v = \omega\sqrt{A^2 - x^2}$$
Max at $x=0$: $v_{max} = A\omega$; zero at $x = \pm A$
Kinetic Energy
$$KE = \frac{1}{2}m\omega^2(A^2 - x^2) = \frac{1}{2}k(A^2-x^2)$$
Potential Energy
$$PE = \frac{1}{2}kx^2 = \frac{1}{2}m\omega^2 x^2$$
Total Mechanical Energy
$$E = KE + PE = \frac{1}{2}kA^2 = \frac{1}{2}m\omega^2 A^2 = \text{constant}$$
Spring–Mass System (horizontal/vertical)
$$T = 2\pi\sqrt{\frac{m}{k}}$$
Two Masses on Spring (Reduced Mass)
$$T = 2\pi\sqrt{\frac{\mu}{k}}, \quad \mu = \frac{m_1 m_2}{m_1+m_2}$$
Series Springs
$$\frac{1}{k_{eq}} = \frac{1}{k_1} + \frac{1}{k_2}$$
Parallel Springs
$$k_{eq} = k_1 + k_2$$
Simple Pendulum
$$T = 2\pi\sqrt{\frac{\ell}{g}} = 2\pi\sqrt{\frac{\ell}{g_{eff}}} \quad\text{(accelerating frame)}$$
$g_{eff}$ = net effective gravity in accelerating reference frame
Compound Pendulum (Physical Pendulum)
$$T = 2\pi\sqrt{\frac{I}{mg\ell}}, \quad I = I_{cm} + m\ell^2$$
$\ell$ = distance between pivot and CM
Torsional Pendulum
$$T = 2\pi\sqrt{\frac{I}{C}}$$
$C$ = torsional constant of the wire
Superposition of SHMs (Same Direction)
$$A = \sqrt{A_1^2 + A_2^2 + 2A_1 A_2\cos\delta}, \quad \tan\phi = \frac{A_2\sin\delta}{A_1 + A_2\cos\delta}$$
Damped Oscillations (Small damping)
$$x = A_0 e^{-bt/2m}\sin(\omega't + \phi), \quad \omega' = \sqrt{\frac{k}{m}-\frac{b^2}{4m^2}}$$
Energy in Damped Oscillation
$$E(t) = \frac{1}{2}kA_0^2\,e^{-bt/m}$$
Quality Factor (Q value)
$$Q = \frac{\omega'}{2\gamma} \quad \text{where } \gamma = \frac{b}{2m}$$
Forced Oscillation (Resonance)
$$A = \frac{F_0/m}{\sqrt{(\omega_0^2 - \omega_d^2)^2 + b^2\omega_d^2/m^2}}$$
At resonance $\omega_d \approx \omega_0$: $A_{max} = \dfrac{F_0}{b\omega_0}$

String Waves

Waves
Wave Equation
$$\frac{\partial^2 y}{\partial t^2} = v^2 \frac{\partial^2 y}{\partial x^2}$$
Travelling Wave
$$y(x,t) = A\sin(\omega t \pm kx + \phi_0)$$
$-kx$: wave in $+x$ direction; $+kx$: wave in $-x$ direction
Speed of Transverse Wave on String
$$v = \sqrt{\frac{T}{\mu}}, \quad \mu = \frac{m}{L}\ \text{(linear mass density)}$$
Wave Number & Phase
$$k = \frac{2\pi}{\lambda} = \frac{\omega}{v}, \quad \Delta\phi = \frac{2\pi}{\lambda}\Delta x = \frac{2\pi}{T}\Delta t$$
Average Power Transmitted
$$P = 2\pi^2 f^2 A^2 \mu v$$
Intensity
$$I = 2\pi^2 f^2 A^2 \rho v$$
Reflected & Transmitted Amplitudes
$$A_r = \frac{k_1 - k_2}{k_1 + k_2}A_i, \quad A_t = \frac{2k_1}{k_1 + k_2}A_i$$
Phase change of $\pi$ on reflection from denser medium
Standing Wave (Fixed Both Ends)
$$f_n = \frac{n}{2L}\sqrt{\frac{T}{\mu}}, \quad n = 1, 2, 3, \ldots$$
$f_1$ = fundamental; $f_n = nf_1$; $n$ = number of loops
Standing Wave (One End Free)
$$f_n = \frac{(2n-1)}{4L}\sqrt{\frac{T}{\mu}}, \quad n = 1, 2, 3, \ldots$$
Only odd harmonics; $f_1 : f_3 : f_5 = 1 : 3 : 5$
Standing Wave Nodes & Antinodes
$$\text{Node–Node} = \frac{\lambda}{2}, \quad \text{Node–Antinode} = \frac{\lambda}{4}$$

Heat & Thermodynamics

Thermal Physics
RMS Speed
$$v_{rms} = \sqrt{\frac{3RT}{M_{mol}}} = \sqrt{\frac{3kT}{m}} = \sqrt{\frac{3P}{\rho}}$$
Average Speed
$$\bar{v} = \sqrt{\frac{8RT}{\pi M_{mol}}} = \sqrt{\frac{8kT}{\pi m}}$$
Most Probable Speed
$$v_p = \sqrt{\frac{2RT}{M_{mol}}} = \sqrt{\frac{2kT}{m}}$$
Speed Comparison
$$v_{rms} : \bar{v} : v_p = \sqrt{3} : \sqrt{\frac{8}{\pi}} : \sqrt{2} \approx 1.73 : 1.60 : 1.41$$
Equipartition of Energy
$$\text{KE per molecule} = \frac{1}{2}fkT$$
$f$ = degrees of freedom; mono=3, di=5, poly=6
Internal Energy (Ideal Gas)
$$U = \frac{f}{2}nRT$$
Specific Heats (Ideal Gas)
$$C_v = \frac{f}{2}R, \quad C_p = \left(\frac{f}{2}+1\right)R$$
Mayer's Relation
$$C_p - C_v = R \quad\text{(ideal gas only)}$$
Adiabatic Index
$$\gamma = \frac{C_p}{C_v} = 1 + \frac{2}{f}$$
Monatomic: 5/3 ≈ 1.67; Diatomic: 7/5 = 1.4; Triatomic: 4/3 ≈ 1.33
Isothermal Work Done
$$W = nRT\ln\!\left(\frac{V_f}{V_i}\right) = 2.303\,nRT\log_{10}\!\left(\frac{V_f}{V_i}\right)$$
Isochoric Process
$$dW = 0, \quad \Delta U = nC_v\Delta T = Q$$
Isobaric Process
$$W = nR(T_f - T_i) = P\Delta V, \quad Q = nC_p\Delta T$$
Adiabatic Relations
$$PV^\gamma = \text{const}, \quad TV^{\gamma-1} = \text{const}, \quad TP^{(1-\gamma)/\gamma} = \text{const}$$
Adiabatic Work Done
$$W = \frac{nR(T_i - T_f)}{\gamma - 1} = \frac{P_1V_1 - P_2V_2}{\gamma - 1}$$
Cyclic Process
$$\Delta Q = \Delta W \quad (\Delta U = 0)$$
Carnot Efficiency
$$\eta = 1 - \frac{T_2}{T_1} = 1 - \frac{Q_2}{Q_1} = \frac{W}{Q_H}$$
Coefficient of Performance (Refrigerator)
$$\beta = \frac{Q_2}{W} = \frac{T_2}{T_1 - T_2}$$
Entropy Change
$$\Delta S = \int_i^f \frac{dQ}{T}, \quad \Delta S_{adiabatic} = 0$$
Mixture of Gases (Molar Quantities)
$$\bar{M} = \frac{n_1 M_1 + n_2 M_2}{n_1+n_2}, \quad C_{v,mix} = \frac{n_1 C_{v1}+n_2 C_{v2}}{n_1+n_2}$$
Thermal Expansion
Linear Expansion
$$\Delta L = \alpha L_0 \Delta T, \quad L = L_0(1 + \alpha\Delta T)$$
Area / Superficial Expansion
$$\Delta A = \beta A_0 \Delta T, \quad \beta \approx 2\alpha$$
Volume / Cubical Expansion
$$\Delta V = \gamma V_0 \Delta T, \quad \gamma \approx 3\alpha$$
Thermal Stress
$$\frac{F}{A} = Y\alpha\Delta T$$
Pendulum Clock (Temperature)
$$\frac{\Delta T_{period}}{T} = \frac{1}{2}\alpha\Delta\theta$$
Heat Transfer
Thermal Conduction (Fourier's Law)
$$\frac{dQ}{dt} = -KA\frac{dT}{dx}$$
Thermal Resistance
$$R_{th} = \frac{\ell}{KA}$$
Stefan-Boltzmann Law
$$u = e\sigma T^4, \quad \sigma = 5.67\times10^{-8}\ \text{W/m}^2\text{K}^4$$
Net Radiation Power
$$P_{net} = e\sigma A(T^4 - T_0^4)$$
Wien's Displacement Law
$$\lambda_m T = b = 2.898\times10^{-3}\ \text{m·K}$$
Newton's Law of Cooling
$$\frac{d\theta}{dt} = -k(\theta - \theta_0), \quad \theta = \theta_0 + (\theta_i - \theta_0)e^{-kt}$$
Kirchhoff's Law
$$\frac{E_{body}}{a_{body}} = E_{blackbody}$$
Calorimetry
Specific Heat
$$S = \frac{Q}{m\Delta T}$$
Molar Specific Heat
$$C = \frac{Q}{n\Delta T}$$

Electrostatics

Electricity
Coulomb's Law
$$\vec{F} = \frac{1}{4\pi\varepsilon_0}\frac{q_1 q_2}{r^2}\hat{r} = k\frac{q_1 q_2}{r^2}\hat{r}$$
$k = 9\times10^9$ N·m²/C²; $\varepsilon_0 = 8.85\times10^{-12}$ C²/Nm²
Electric Field (General)
$$\vec{E} = \frac{\vec{F}}{q_0} = -\nabla V$$
Gauss's Law
$$\phi_E = \oint \vec{E}\cdot d\vec{A} = \frac{q_{enc}}{\varepsilon_0}$$
Electric Potential
$$V_P = \frac{W_{\infty\to P}}{q_0} = \frac{kq}{r}$$
Potential Difference
$$V_A - V_B = -\int_A^B \vec{E}\cdot d\vec{r}$$
Energy Density
$$u = \frac{1}{2}\varepsilon_0 E^2$$

Field on Axis (End-on)

Torque in Uniform Field

Error & Measurement

Measurement
Absolute Error
$$\Delta a_n = |a_{mean} - a_n|$$
Mean Absolute Error
$$\overline{\Delta a} = \frac{1}{n}\sum_{i=1}^n|\Delta a_i|$$
Relative Error
$$\delta_r = \frac{\overline{\Delta a}}{a_{mean}}$$
Percentage Error
$$\delta\% = \frac{\overline{\Delta a}}{a_{mean}}\times 100\%$$
Error Propagation (Sum/Difference)
$$\text{If } z = x \pm y: \quad \Delta z = \Delta x + \Delta y$$
Error Propagation (Product/Quotient)
$$\text{If } z = x^a y^b z^c: \quad \frac{\Delta z}{z} = a\frac{\Delta x}{x} + b\frac{\Delta y}{y} + c\frac{\Delta w}{w}$$
Least Count (Vernier Callipers)
$$LC = 1\ \text{MSD} - 1\ \text{VSD}$$
Screw Gauge Reading
$$\text{Reading} = \text{MSR} + (CSR)\times LC$$
$LC = \text{pitch}/\text{no. of circular scale divisions}$; pitch = advance per rotation
Significant Figures Rules
RuleExample
All non-zero digits are significant$152.4$ → 4 SF
Zeros between non-zeros are significant$1002$ → 4 SF
Leading zeros (after decimal, before non-zero) are NOT significant$0.0023$ → 2 SF
Trailing zeros with decimal point ARE significant$1.500$ → 4 SF
Trailing zeros without decimal point are NOT significant$1500$ → 2 SF

Principle of Communication

Electronics
Range of Transmission Tower
$$d_T = \sqrt{2Rh_T}, \quad d_M = \sqrt{2Rh_T}+\sqrt{2Rh_R}$$
$R$ = Earth's radius; $h_T$, $h_R$ = height of transmitter/receiver
Amplitude Modulation Index
$$m_a = \frac{A_m}{A_c} = \frac{A_{max}-A_{min}}{A_{max}+A_{min}}$$
AM Signal
$$c_m(t) = A_c\sin\omega_c t + \frac{\mu A_c}{2}\cos(\omega_c-\omega_m)t - \frac{\mu A_c}{2}\cos(\omega_c+\omega_m)t$$
Sidebands
$$f_{USB} = f_c + f_m, \quad f_{LSB} = f_c - f_m, \quad \text{BW} = 2f_m$$
Total Modulated Index (multiple signals)
$$m_t = \sqrt{m_1^2 + m_2^2 + m_3^2 + \cdots}$$
Power in AM Waves
$$P_c = \frac{A_c^2}{2R}, \quad P_{sb} = \frac{m_a^2 A_c^2}{4R}, \quad P_T = P_c\!\left(1+\frac{m_a^2}{2}\right)$$
FM — Frequency Deviation & Index
$$\Delta f = f_{max}-f_c = k_f E_m, \quad m_f = \frac{\Delta f}{f_m}$$
Current Ratio (AM)
$$\frac{I_t}{I_c} = \sqrt{1+\frac{m_a^2}{2}}$$
